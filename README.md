## Projects

### 🔹 IT Spend Intelligence & Optimization
**MySQL · Power BI · DAX**

![Executive Overview](https://raw.githubusercontent.com/Grandlad/Portfolio-Projects/main/IT%20Spend%20Intelligence%20%26%20Optimization%20(SaaS%2C%20Cloud%20%26%20Software%20Assets)/PowerBI/Screen_Shots/IT%20Spend%20Intelligence%20-%20Executive%20Overview.png)

An enterprise-grade IT Asset & Cost Management platform covering the full lifecycle of IT spend governance — from raw data modelling in MySQL to interactive Power BI dashboards.

| Module | Description |
|---|---|
| **Data Foundation** | TBM taxonomy, CapEx/OpEx classification, Shadow IT detection |
| **HAM** | Hardware lifecycle tracking, warranty risk, offboarding gap detection |
| **SAM** | License utilization, shelfware detection, renewal radar |
| **FinOps** | Cloud burn rate, EOY forecasting, untagged cost audit |
| **Procurement** | Contract expiry radar, vendor concentration, cashflow analysis |
| **Governance** | Database-level triggers, contract alerting view |

**Key stats:** 38M PLN spend analysed · 8-table star schema · 7 SQL modules · 6 Power BI report pages · 20+ DAX measures
